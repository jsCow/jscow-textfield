jsCow.res.components.textfield Class

Defined in: src\jscow\components\jscow-textfield.js:1

jsCow textfield component is a reusable component used in client side jsCow applications. A textfield is a component, which enables users to interact with an application.

    textfield6 = jsCow.get(jsCow.res.components.textfield, {
        value: 'Textfield with info status',
